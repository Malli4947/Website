Drop your Logistics hero background video here as:

    logistics-bg.mp4

Recommended: short, silent, looping clip (cargo ships / port / aircraft),
1080p or 720p, ~10–20s, H.264 MP4, a few MB.

Until a file is present, the Logistics hero shows an animated cargo-scene
fallback, so the page always looks complete.

To change the path, edit divisionExtras.logistics.hero.video in
src/data/content.js
